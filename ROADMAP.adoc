== Phase 1: Foundation (v0.1.0 - v0.5.0)

=== v0.1.0 "Genesis" -- Initial Alpha
*Status:* CURRENT

==== Deliverables
* [x] Ada 2022 project structure
* [x] Basic ncurses TUI framework
* [x] Category-based option organization
* [x] Static hidden option database
* [x] GNAT project file with build modes
* [x] Alire package manifest
* [x] README.adoc documentation
* [x] ROADMAP.adoc (this document)

==== Technical Specifications
* Ada 2022 standard compliance
* ncursesada dependency
* Debug/Release/SPARK build modes
* Cross-platform directory structure

==== Quality Gates
* Compiles without warnings (debug mode)
* Basic smoke test passes
* Documentation complete

---

=== v0.2.0 "Discovery" -- Option Detection Engine

==== Deliverables
* [ ] Parse `nano --help` output dynamically
* [ ] Detect nano version from binary
* [ ] Scan `/usr/share/nano/*.nanorc` for syntax files
* [ ] Parse existing `~/.nanorc` configuration
* [ ] Build runtime option database
* [ ] Categorize options by functionality
* [ ] Mark hidden/undocumented options

==== New Features
* Dynamic option discovery based on installed nano version
* Support for nano versions 2.0 through 8.x
* Option availability tracking by version
* Syntax file enumeration

==== Technical Tasks
* [ ] Implement `Nano_Aider.Discovery` package
* [ ] Add subprocess execution for `nano --help`
* [ ] Create option database record types
* [ ] Implement version comparison logic
* [ ] Add unit tests for discovery

---

=== v0.3.0 "Navigator" -- Full TUI Navigation

==== Deliverables
* [ ] Complete keyboard navigation (vim-style + arrows)
* [ ] Search functionality with highlighting
* [ ] Option detail view with full descriptions
* [ ] Scroll indicators and position tracking
* [ ] Mouse support (optional)
* [ ] Window resize handling
* [ ] Help overlay system

==== New Features
* Fuzzy search across all options
* Option value preview
* Breadcrumb navigation
* Quick jump to category

==== Technical Tasks
* [ ] Refactor TUI into window management system
* [ ] Implement search algorithm with ranking
* [ ] Add panel support for overlays
* [ ] Handle terminal resize signals
* [ ] Optional mouse event processing

---

=== v0.4.0 "Scribe" -- Configuration Export

==== Deliverables
* [ ] Export to `.nanorc` format
* [ ] Export to micro `settings.json`
* [ ] Export to Markdown documentation
* [ ] Import existing configuration
* [ ] Diff view between current and exported
* [ ] Backup before overwrite protection

==== New Features
* One-click configuration export
* Configuration validation before save
* Comment preservation during import/export
* Export with documentation comments

==== Technical Tasks
* [ ] Implement `Nano_Aider.Export` package
* [ ] Create configuration serializers
* [ ] Add file validation routines
* [ ] Implement backup mechanism

---

=== v0.5.0 "Chameleon" -- Profile Management

==== Deliverables
* [ ] Built-in profiles (Minimal, Developer, Writer, Power User)
* [ ] User-defined profile creation
* [ ] Profile switching with preview
* [ ] Profile import/export
* [ ] Profile comparison tool

==== New Features
* Quick profile switching (F1-F4 keys)
* Profile inheritance (extend from base)
* Conditional profiles (by file extension)
* Profile recommendations

==== Technical Tasks
* [ ] Implement `Nano_Aider.Profiles` package fully
* [ ] Create profile storage format
* [ ] Add profile validation
* [ ] Implement comparison diffing

---

=== Release Cadence

* **Alpha releases (0.x):** Every 4-6 weeks
* **Minor releases (x.Y.0):** Every 3-4 months
* **Patch releases (x.y.Z):** As needed for security/bugs
* **Major releases:** ~12-18 months

== Contributing to the Roadmap

This roadmap is a living document. Contributions are welcome:

1. **Feature Requests:** Open a GitHub issue with `[ROADMAP]` prefix
2. **Version Proposals:** Submit a PR modifying this document
3. **Priority Feedback:** Comment on existing roadmap issues

=== Voting on Features

Features are prioritized based on:

* Community votes (thumbs up on issues)
* Maintainer capacity
* Technical dependencies
* Security implications
